## Purpose
This project is an online appointment system designed to help patients book doctor appointments easily, reducing long queues at hospitals.

## Product Scope
The system allows patients to book, cancel, and check appointment statuses from home. Admins and managers can manage doctors and clinic assignments efficiently.

## Technologies Used
- HTML5, CSS3, JavaScript
- PHP (Backend)
- MySQL (Database)
- GitHub Actions & Vercel (CI/CD and Deployment)
